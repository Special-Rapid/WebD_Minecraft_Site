# Minecraft Fan Site

マインクラフトの魅力を紹介するファンサイトです。サバイバル、PVP、建築など、様々なプレイスタイルを紹介しています。

## 📝 概要

このプロジェクトはMinecraftの異なるプレイモードの特徴や楽しさを紹介するウェブサイトです。

## 🎮 セクション

- **Survival** - 資源を収集し、家を建て、モンスターと戦うことで生き延びるゲーム
- **PVP** - プレイヤー同士が戦うモードで、戦略とスキルが求められます
- **Build** - クリエイティブな側面を楽しむモードで、自由に建物や構造物を作成できます

## 🛠️ 技術構成

- HTML5
- CSS3
- Responsive Design（レスポンシブ対応）
- Google Fonts（Pixelify Sans）

## 📂 ファイル構成

```
WebD_Minecraft_Site/
├── index.html      # メインのHTMLファイル
├── style.css       # スタイルシート
├── README.md       # このファイル
├── img/            # 画像ファイル
├── img_idea/       # アイデア画像
└── videoproject/   # ビデオプロジェクト関連
```

## 🎨 特徴

- ビデオバックグラウンドを使用したダイナミックなファーストビュー
- スムーズなスクロール体験
- モバイルレスポンシブ対応
- グラデーションエフェクトによる美しい視覚表現

## 📱 対応デバイス

- PC（1200px以上）
- タブレット・モバイル（786px以下）
