# Minecraft Fan Site

https://special-rapid.github.io/WebD_Minecraft_Site/

マインクラフトの魅力を紹介するファンサイトです。サバイバル、PVP、建築など、様々なプレイスタイルを紹介しています。

## 📝 概要

このプロジェクトはMinecraftの異なるプレイモードの特徴や楽しさを紹介するウェブサイトです。

## 🎮 セクション

- **Survival** - 資源を収集し、家を建て、モンスターと戦うことで生き延びるゲーム
- **PVP** - プレイヤー同士が戦うモードで、戦略とスキルが求められます
- **Build** - クリエイティブな側面を楽しむモードで、自由に建物や構造物を作成できます

## 🛠️ 技術構成

- HTML5
- CSS3
- JavaScript（jQuery）
- ProgressBar.js（ローディングプログレスバー）
- Responsive Design（レスポンシブ対応）
- Google Fonts（Pixelify Sans）
